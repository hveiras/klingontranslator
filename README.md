# Klingon Translator

Translate klingon to its hex represenation and show the species for the given charated usin [stapi](http://stapi.co)

## Requeriments

> node 8.x +

## Setup

> $ git clone https://github.com/hveiras/klingontranslator.git
$ cd klingontranslator
$ npm install
$ chmod +x translate

## Usage

`$ ./translate Uhura`
